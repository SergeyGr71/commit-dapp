# Commit dApp 🚀

Simple onchain commitment tracking app built using Canopy Templates and AI.

## 🔥 Features
- Create commitments
- Track status (pending / completed / failed)
- Simple UI with buttons
- REST API backend

## 🛠 Tech
- Node.js (Express)
- Vanilla JS (Frontend)
- Canopy Templates concept

## ▶️ How to run

npm install
node server.js

Open index.html

## 💡 Idea

This app allows users to create commitments and track whether they complete or fail them — simple but powerful accountability system.

## ⚡ Built fast with AI

Created quickly using AI assistance to demonstrate rapid time-to-market with Canopy approach.
